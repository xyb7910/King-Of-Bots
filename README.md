# King-of-Bots
## 项目简介
🚪适合入门的，简单的，逻辑性较强的`SpringBoot` + `SpringCloud` + `微服务` 训练项目。
## 技术圈
```
SpringBoot 3.0
SpringCloud
MyBaitsPlus
mysql 8.0
Vue3
WebSocket
...
```
其中，技术难点有锁机制的使用，前后端通信，用户信息加密，Alpha搜索等。

## ⏬克隆与启动
```
git clone git@github.com:xyb7910/King-of-Bots.git
cd King-of-Bots
```

## 💻安装与启动

```
npm install
vue ui
```
